# Institution/Workplace Logos

Place logo image files in this directory. The Timeline component will automatically
use these logos when available, and fall back to initials badges if a file is missing.

## Expected Files

| File         | Institution / Organization                        |
|--------------|--------------------------------------------------|
| `diu.png`    | Dhaka International University (DIU)              |
| `mpi.png`    | Metro Police Ideal Law College, Dhaka             |
| `glc.png`    | Gazipur Law College, Gazipur                      |
| `aub.png`    | Asian University of Bangladesh (AUB)              |
| `tguc.png`   | Tongi Govt. University & College, Gazipur         |
| `mhs.png`    | Maligram High School, Bhanga, Faridpur            |
| `sco.png`    | Supreme Court of Bangladesh                       |
| `msc.png`    | Metro Sessions Court, Dhaka                       |

## Format
- Recommended: PNG or SVG with transparent background
- Recommended size: at least 128×128 px
- The component renders them at 48×48 px with `object-contain`
